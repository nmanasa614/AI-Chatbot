Advanced Multimodal AI Chatbot

**Overview

Advanced Multimodal AI Chatbot is a Generative AI project built using Streamlit, Google Gemini AI, SQLite, and Python. The chatbot supports conversational AI, memory management, document understanding, image analysis, report generation, translation, email drafting, and meeting summarization.
**Features

**Core Features

* Gemini AI Integration
* Conversational Chat Interface
* Context-Aware Responses
* Persistent Memory using SQLite
* Chat History Storage
* New Chat Support
* Clear Chat Functionality

**Document & Image Processing

* PDF Upload and Question Answering
* Image Upload and Analysis
* Multimodal Understanding (Text + PDF + Image)

**AI Productivity Tools

* Email Drafting
* Meeting Summarization
* AI Report Generation
* Multilingual Chat Support
* Language Translation

*Additional Features

* Chat Export
* Streamlit Web Interface
* Modular Project Structure
* Environment Variable Support (.env)
